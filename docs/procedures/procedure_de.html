Vorgehensweise:

  1. Legen Sie fest, ob Sie eine Offene-Posten-Liste Ihrer Debitoren oder Kreditoren auswerten möchten.

  2. Geben Sie flexibel an, welches Zahlenformat (z. B. Tausender- oder Dezimaltrennzeichen), welches Datumsformat und welches Trennzeichen Ihre CSV-Datei verwendet. So sind Sie unabhängig vom Exportformat Ihres Buchhaltungsprogramms.

  3. Möchten Sie wissen, welche Rechnungen seit 30, 60 oder 90 Tagen fällig sind? Entscheiden Sie selbst, welche Zeitintervalle Sie anzeigen möchten. Die einzelnen Abschnitte können frei ausgewählt werden.

  4. Laden Sie dann einfach eine Liste der ausstehenden Forderungen bzw. Verbindlichkeiten in die App hoch. Diese muss den Namen der Debitoren bzw. Kreditoren, den jeweiligen Rechnungsbetrag und das Fälligkeitsdatum enthalten. Alternativ können Sie Beispieldaten generieren, um die Funktionen der App zu testen.

  5. Sehen Sie sich nun die pro Debitor oder Kreditor aggregierten Rechnungsbeträge an, die automatisch den von Ihnen definierten Fälligkeitsintervallen zugeordnet wurden. Überprüfen Sie Ihre individuelle Fälligkeitsstruktur in verschiedenen Diagrammtypen. Wer ist Ihr größter Debitor oder Kreditor? Grafische Auswertungen von Konzentrationen erleichtern die Analyse.

  6. Exportieren Sie die Fälligkeitsstruktur-Tabelle, um die Informationen in anderen Anwendungen weiterzuverarbeiten.
